# **App Name**: AdharPrint Pro

## Core Features:

- Manual Adhar Card Entry: Allows users to manually enter Adhar card details into the system.
- Advanced Formatting Options: Provides advanced options for formatting the Adhar card print, including font size, alignment, and layout adjustments.
- Image Upload: Enables users to upload a photo for the Adhar card, with options for resizing and cropping.
- QR Code Generation: Generates a QR code based on the Adhar card details.
- Preview and Download: Provides a preview of the Adhar card print and allows users to download it in PDF format.
- AI powered content generation: When the confidence in character recognition is low, LLM-powered tool enables refinement and completion of partially complete name or address field, improving user experience when image isn't 100% sharp and clear

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to inspire confidence and security.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean layout.
- Accent color: Teal (#008080) for interactive elements and highlights, adding a touch of sophistication.
- Font pairing: 'Inter' sans-serif for headings, and 'PT Sans' sans-serif for body text. Ensures readability and modern design.
- Use simple, professional icons for key functions such as upload, download, and print.
- Clean, user-friendly layout with a focus on ease of data entry and clarity of preview.
- Subtle transitions and loading animations to enhance user experience and provide feedback during processing.
